# Workshop INCTEECbio: Versionamento, Git e GitHub

Este repositório contém o material do workshop sobre versionamento de código, Git e GitHub, organizado para o INCTEECbio.

## Materiais Disponíveis

-   **Programação Workshop Bioinformática.pdf**: Programação detalhada do workshop, incluindo horários e tópicos abordados.
-   **workshop_git_manha_LBB_Rhewter.pdf**: Apresentação utilizada na parte da manhã, cobrindo conceitos fundamentais de versionamento, Git e GitHub.

## Objetivo

Capacitar os participantes a: - Entender os conceitos de versionamento de código. - Utilizar o Git para controle de versões local. - Colaborar em projetos utilizando o GitHub.

## Como Usar Este Repositório

1.  Baixe ou clone este repositório:

    ``` sh
    git clone https://github.com/Rhewter/Workshop_INCTEECbio_Git_e_GitHub.git
    ```

2.  Acesse os arquivos PDF para acompanhar o conteúdo apresentado no workshop.
